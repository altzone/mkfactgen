# mkfactgen
Generation de facture VOIP excel et stdout
Utilisation:
 mkfactgen.php id_mera num_facture type_facture forfait type_sortie

 Type Facture:
  - 0 (Standard)
  - fixe (Illimité fixe)
  - fixemob (Illimité fixe & mobile)

 Forfait:
  - 0 (pas de forfait)
  - chiffre en heure du forfait

 Type_sortie:
  - 0 (sortie standard)
  - json (sortie JSON)
